## API Reference

#### Get all items

```http
  GET /all
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `api_key` | `string` | **NOT Required**. |

#### Get item

```http
  GET /${name}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `name`      | `string` | **Required**. name of platform to fetch |
